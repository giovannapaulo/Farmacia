Sistema de Gerenciamento de Farmácia - MicrosserviçosProjeto desenvolvido para a rede de farmácias com o objetivo de refatorar o sistema monolítico para uma arquitetura distribuída de microsserviços.  🏗️ Arquitetura do SistemaO sistema foi projetado em duas versões para demonstrar a evolução de comunicação entre serviços:Versão 1: Comunicação Síncrona baseada em REST (HTTP).  Versão 2: Comunicação Assíncrona baseada em Mensageria com RabbitMQ para maior resiliência e desacoplamento.  🛠️ Tecnologias UtilizadasJava 17+Spring Boot (Web, JPA, AMQP)   MySQL (Banco de Dados Relacional)   RabbitMQ (Message Broker para mensageria)   📋 Funcionalidades PrincipaisCadastro de medicamentos e produtos de higiene.  Vendas via balcão ou internet com cálculo automático de comissão.  Gestão de descontos inteligentes (Convênio, Bonificações e Idoso).  Emissão de NF com CPF.  Controle de medicamentos controlados com notificação de receitas.  Relatórios gerenciais (Vendas por período, Estoque, etc.).
